# Patronus Exchange

## Available API:

- USER CRUD
- BEARER authentication
- Create Order
  - Market order
  - Limit order
- Order is executed automatically on creation if matches found
- Transaction management
- Ethereum integration


## Setup Guide

### Prerequisites:
1. MongoDB: 4.4.3
2. Redis: 7.0.11
3. Go: 1.21.3

### Build Environment:
Goland 2023 Ultimate Edition
